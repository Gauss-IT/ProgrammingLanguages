Historia

Microsoft announced Small Basic in October 2008,[1] and released the first stable version for distribution on July 12, 2011,[17] on a Microsoft Developer Network (MSDN) website, together with a teaching curriculum[19] and an introductory guide.[20] Between announcement and stable release, a number of Community Technology Preview (CTP) releases were made. On March 27, 2015, Microsoft released Small Basic version 1.1,[18] which fixed a bug and upgraded the targeted .NET Framework version from version 3.5 to version 4.5. Microsoft released Small Basic version 1.2 on October 1, 2015.[3] Version 1.2 was the first update after a four-year hiatus to introduce new features to Small Basic. The update added classes for working with Microsoft's Kinect motion sensors,[3] increased the number of languages supported by the included Dictionary object, and fixed a number of bugs.[4] On February 19, 2019, Microsoft announced Small Basic Online (SBO). It is open source software released under MIT License on GitHub.[21][22]

Gjuhët

Në Small Basic, dikush shkruan ilustrimin "Përshëndetje, Botë!" program si më poshtë: TextWindow.WriteLine ("Përshëndetje, Botë!") Microsoft Small Basic është në përfundim i plotë. Mbështet degëzimin me kusht, strukturat e lakut dhe nënrutat për trajtimin e ngjarjeve. Variablat janë shtypur dobët dhe dinamikë pa rregulla të parashikimit. Degëzimi me kusht Shembulli i mëposhtëm tregon degëzimin me kusht. Kërkon temperaturën aktuale në Fahrenheit dhe komenton përgjigjen.
