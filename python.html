Gjuha Programuese Python

Python është një gjuhë programimi e interpretuar, e nivelit të lartë, me qëllim të përgjithshëm. Eshtww krijuar nga Guido van Rossum dhe wwshtww lëshuar nww pwwrdorim për herë të parë në vitin 1991. Filozofia e dizajnit të Python thekson lexueshmërinë e e kodit me përdorimin e tij të dukshëm të hapësirës së bardhë pwwr tww shprehur strukturwwn e programit. Ndërtimet gjuhësore të saj dhe qasja e orientuar nga objekti synojnë të ndihmojnë programuesit të shkruajnë kod të qartë, logjik për projekte të vogla dhe të mëdha.

Python është shtypur dinamikisht dhe mblidhet mbeturina. Ai mbështet paradigma të shumta programimi, përfshirë programimin procedural, të orientuar drejt objektit dhe funksionimit. Python është përshkruar shpesh si një gjuhë "përfshirë bateritë" për shkak të bibliotekës standarde të plotë. [28]

Python u konceptua në fund të viteve 1980 si një pasardhës i gjuhës ABC. Python 2.0, lëshuar 2000, prezantoi karakteristika si kuptimi i listës dhe një sistem i mbledhjes së mbeturinave të afta për të mbledhur cikle referimi. Python 3.0, lëshuar në vitin 2008, ishte një rishikim i madh i gjuhës që nuk është plotësisht i përputhshëm me prapambetje, dhe shumë kodi Python 2 nuk funksionon i pandryshuar në Python 3. Për shkak të shqetësimit për sasinë e kodit të shkruar për Python 2, mbështetje për Python 2.7 (lëshimi i fundit në serinë 2.x) u shtri në vitin 2020. Zhvilluesi i gjuhës Guido van Rossum mori përgjegjësinë e vetme për projektin deri në korrik 2018, por tani ndan drejtimin e tij si anëtar i një këshilli drejtues pesë-personësh. [29] [30] [31]

Gjuha e Python 2, d.m.th. Python 2.7.x, është "perëndimi i diellit" në 1 Janar 2020, dhe ekipi i vullnetarëve të Python nuk do të rregullojë çështjet e sigurisë, ose do ta përmirësojë atë në mënyra të tjera pas kësaj date. [32] [33] Me fundin e jetës, vetëm Python 3.6.x dhe më vonë do të mbështetet. [34]

Përkthyesit e Python janë në dispozicion për shumë sisteme operative. Një bashkësi globale e programuesve zhvillon dhe mirëmban CPython, një zbatim i referencës me burim të hapur [35]. Një organizatë jofitimprurëse, Fondacioni Python Software, administron dhe drejton burimet për zhvillimin e Python dhe CPython.
